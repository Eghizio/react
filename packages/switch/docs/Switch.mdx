# Switch

This component allows users to toggle between validity for a condition.

<PackageInfo packageData={packageData} componentNames={['Switch']} />

## Examples

```jsx example
<Switch value={false} />
```

```jsx example
<Switch value={true} />
```

```jsx example
<Switch disabled />
```

### Writing your own custom click handler

```jsx
const [value, setValue] = useState(false);

<Switch
    value={value}
    onClick={() => {
        setValue(!value);
        alert('Your own custom action when toggled');
    }}
/>;
```

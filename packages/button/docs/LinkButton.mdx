# LinkButton

A button disguised as a link.

<PackageInfo packageData={packageData} componentNames={['LinkButton']} />

## Example

```jsx example
<LinkButton>Link like button</LinkButton>
```

## Use with care

**NOTE: Do not use LinkButton to navigate to another page, use a normal link
instead.**

Users expect links to behave like links. For instance, they should be able to
open them in new tabs, copy the link location etc. Be aware of this when using
the LinkButton. A better option might be using a flat varianted
[Button](/Button) instead.

## Accessibility

If the button doesn't have visible text content, such as when used with only an
icon, an `aria-label` prop must be provided for accessibility.

## Visual options

### Small

# Fabric React

Home to the React implementation of
[Fabric](https://pages.github.schibsted.io/finn/fabric-design/), FINN's design
system.

## Under construction

Fabric React is currently in active development. We're gradually porting
components from
[Troika React](https://pages.github.schibsted.io/finn/troika-react/).

In the meantime, it should be possible to run Troika React components on a
Fabric page.
